CHANT
=====

the minimum, casual, tiny and easy closed chat.

tl;dr
=====

Hit it now, by using your cool appengine name.

```zsh
goapp deploy -application your-appengine-appname ./app
```

Develop
===========

```
npm start & goapp serve app
```

Release
==========

```
npm run release && \
goapp deploy -application your-appengine-appname ./app
```
